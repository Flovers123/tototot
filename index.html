Проект

О программе

Программа Motocikl Korporetion создана для административной части сайта по продаже мотоциклов. Упрощённая программа для работы с покупками, магазином, и просмотром товаров.

  • 1
    Функционал программы

    В программе реализованы возможности покупки мотоцикла со стороны пользователя, добавления покупок в базы данных. Со стороны администратора есть возможность добавления товара, просмотра прибыли, просмотра корзины пользователя, редактирования личных данных, добавления скидок на имеющиеся товар.

  • 2
    Немного о программе

    Программа написана я языке: C# WinForms
    Базы данных: MySql

Коды программы

Назначения кодов

1) Код авторизации

Позволяет зайти в аккаунт программы.

2) Код регистрации

Позволяет зарегистрироваться в аккаунт программы.

3) Код покупки мото

Позволяет пользователю купить мотоцикл.

4) Код административной панели

Позволяет администратору пользоваться программой.

5) Код добавления мото

Позволяет администратору добавить новый мотоцикл.

6) Код добавления информации о мото

Позволяет администратору добавить иноформацию о новом мотоцикле.

7) Код добавления информации о внутрнности мото

Позволяет администратору добавить иноформацию о новом мотоцикле.

8) Код добавления скидки

Позволяет администратору добавить скидку на мотоцикл.

9) Код личных данных администратора

Позволяет администратору редактировать свои личные данные в приложении.

10) Код корзины покупок

Позволяет администратору посмотреть корзину покупок пользователей.

11) Код добавления нового администратора

Позволяет администратору добавить нового администратора в программе.

12) Код информации о имеющемуся товаре

Позволяет администраторупосмотреть информацию о имеющемя товаре.

13) Код подсчёта прибыли

Позволяет администратору посмотреть всю прибыль купленных мотоциклов.
